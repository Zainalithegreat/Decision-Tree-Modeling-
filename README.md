This project applies supervised machine learning (Decision Tree Classification) to analyze mental health and social media datasets. The goal is to build predictive models that classify outcomes and uncover patterns in user behavior.

Features
- Data preprocessing and feature selection
- Separation of features (X) and labels (y)
- Train/test dataset splitting
- Decision Tree modeling using Scikit-learn
- Data visualization using histograms, box plots, and scatter plots
- Model evaluation using confusion matrices and accuracy metrics
- Decision tree visualization for model interpretability

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
